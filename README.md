# Coffee shop - frontend

Here you can buy really delicious coffee.

## Tech stack:

- TypeScript, Next
- SASS, TailwindCSS, CSSModules
- ESlint, Prettier

## Project launch:

Clone repository:

```bash
   git clone https://github.com/aanndrewss/coffee-shop-frontend.git
```

Install packages:

```bash
      npm i
```

Running the app:

```bash
     npm run start
```
